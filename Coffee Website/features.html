Website Features

This page lists the main features implemented on the Hot Grind Coffee Roasters website, with links to where you can experience them.

  • About Us Section - Introduction to our brand and specialty coffee.
  • User Registration - Create an account to receive special offers and updates.
  • Order Placement Form - Choose pick-up or delivery and payment methods with dynamic form fields.
  • Conditional Delivery Address - Delivery address field appears only when 'Delivery' is selected.
  • Conditional Payment Details - Credit card details appear only when 'Pay Online' is selected.
  • Menu Showcase - Display of specialty coffee products with images and descriptions.
  • Customer Reviews - Testimonials from happy customers.
  • Responsive Design - The website adapts for mobile and desktop screens.
  • Navigation Highlighting - The current page link is automatically highlighted for better user experience.
  • Billing Address Autofill - Automatically copies the delivery address into the billing address field when selected.
  • Card Type-Specific Length - Credit card number field enforces correct length based on selected card type (e.g., AMEX = 15 digits).
